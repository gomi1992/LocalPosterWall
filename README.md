# 本地电影海报墙

一个简洁的本地电影管理工具，以海报墙形式展示本地电影库。

## 功能特点

- 海报墙展示：以网格形式展示电影海报
- 自动识别：自动识别电影标题、年份和分辨率
- 智能排序：支持按年份和标题排序
- 播放器集成：支持自定义视频播放器
- 中文优化：完整的中文界面，支持中文标题排序

## 使用说明

1. 安装依赖：
```bash
pip install -r requirements.txt
```

2. 运行程序：
```bash
python movie_wall.py
```

3. 初始配置：
   - 点击"管理文件夹"添加电影文件夹
   - 点击"选择播放器"配置视频播放器
   - 点击"刷新列表"更新电影列表

4. 使用说明：
   - 点击海报可直接播放电影
   - 使用顶部的排序选项可以调整电影排序方式
   - 支持按年份和标题排序

## 文件夹结构要求

电影文件夹命名格式：`电影名称 (年份)`
例如：`红色一号：冬日行动 (2024)`

文件夹内需包含：
- 海报图片（命名为 poster）
- 视频文件（支持常见视频格式）
- 视频命名（示例：蝙蝠侠：黑暗骑士 (2008) 2160p.mkv）

## 项目结构

```
movie_wall/
├── movie_wall.py      # 主程序
├── config_manager.py  # 配置管理
├── movie_scanner.py   # 电影扫描
├── poster_wall.py     # 海报墙组件
├── requirements.txt   # 依赖列表
└── README.md         # 项目说明
```

## 依赖项

- PySide6：GUI框架
- python-vlc：视频播放支持
- pillow：图片处理
- opencv-python：视频信息提取
- pypinyin：中文拼音处理

## 系统要求

- Python 3.8+
- Windows 10/11
